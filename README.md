#Question 1 : Add  Two Number


Given two values, they can be integer or floating point numbers, add them.

Input Format

Two values separated by a space.

Constraints

No constraints.

Output Format

One value, the result of sum of two input values.

Sample Input 0

5 4
Sample Output 0

9
Explanation 0

5 + 4 = 9
